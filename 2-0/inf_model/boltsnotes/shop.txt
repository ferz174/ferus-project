Примечания:
1. Технические требования к высокопрочным болтам, гайкам и шайбам по ГОСТ Р 52643-2006.
2. Материал: болты из стали 40Х по ГОСТ4543-71* с наименьшим временным сопротивлением 110 кг/мм2;
   гайки из стали Ст35 или Ст40 по ГОСТ 1050-88*; шайбы из стали Ст5пс по ГОСТ 380-2005.
3. Заказано количество гаек по 1 шт., шайб - по 2 шт. на каждый болт.
4. Количество метизов заказано без надбавки или с 1% надбавкой.
5. По данному списку  завод метизы поставляет или не поставляет. 
Возможные примечания
2. Материал:  болты - сталь 40Х по ГОСТ 4543-71*, с суженным содержанием углерода -
  от 0,37% до 0,42%; гайки - сталь 35, 40, 35Х, 40Х по ГОСТ 1050-88*; шайбы - стали,
  рекомендованные ГОСТ Р 52643-2006 для изготовления шайб с твердостью НРС
  от 35 до  42 единиц. 
3. Дополнительные требования к поставке высокопрочных болтов: контроль на соответствие
  эталону микроструктуры, испытание на определение обезуглероживания (науглероживание),
  исполнение ХЛ.
